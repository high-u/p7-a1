  │ 期間 │ 一つの会話の間         │ 永続                         │
  ├──────┼────────────────────────┼──────────────────────────────┤
  │ 粒度 │ ターン単位の全文       │ 抽出・要約された事実・出来事 │
  └──────┴────────────────────────┴──────────────────────────────┘

  - 100ターン超えると LLM のコンテキストが溢れる、という技術的制約への対処

  「途切れた日付またぎはまっさらなNewセッション」 → アイドルタイム超過で自然に実現します
